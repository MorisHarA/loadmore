# react-loadmore

A loadmore component for react.

## Usage

`npm install react-loadmore --save`

```xml
<LoadMore
  loading={loading}
  completed={completed}
  onLoadMore={this.handleLoadMore}
>
  <WrappedComponent />
</LoadMore>
```

## All props

PropTypes

- **loading**: `PropTypes.bool`
- **completed**: `PropTypes.bool`
- **onLoadMore**: `PropTypes.func.isRequired`
